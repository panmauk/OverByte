# OverByte

A simple Windows desktop app that inflates files to a target size.

## What it does

OverByte lets you take any file and increase its size to whatever you want. Pick a file, set a target size, and it pads the file to match. Works on single files or entire folders.

## Modes

- **Single File** - Browse for a file, set a target size, and inflate it.
- **Folder** - Inflate all files in a folder at once, or compress and enlarge them.

## Requirements

- Windows (x64)
- .NET 10

## Usage

Run `FileInflator.exe` from the `publish` folder. No install needed -- it's a self-contained single-file app.
